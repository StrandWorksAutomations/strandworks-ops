is kept up-to-date, but it is ambiguous whether Supabase or git is the primary record.

   or is Supabase the primary store of record for qa_* data?
2. If Supabase is a new authoritative data store, does the owner want to update VISION/SPEC via
   Intake to sanction this pattern?
