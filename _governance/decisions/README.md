# Decisions queue

One file per pending decision: `YYYY-MM-DD-<slug>.md` with frontmatter
(id, filed, filed-by, question) + context + options. Agents FILE decisions
here; only the owner RULES (token typed in a session, or — once the cockpit
ships — a button tap that commits `ruling`, `ruled`, `source`). Ruled files
move to `resolved/`. Rulings here carry the same authority as typed tokens.
