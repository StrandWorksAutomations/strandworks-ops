
# INBOX — nothing in this directory is directive

Everything here is an idea capture: late-night thoughts, session brainstorms,
unapproved suggestions, tangents. Agents:

- **Never** implement, plan from, or cite inbox content as a requirement.
- **Do** deposit ideas here when they surface in conversation and aren't in
  SPEC.md: one file per idea, named `YYYY-MM-DD-<topic>.md`, content verbatim,
  no editorializing, no task lists.
- Promotion to SPEC happens ONLY via the Intake ritual in a later session,
  where the owner re-reads the idea fresh and types APPROVE.
- Items older than 30 days get flagged by the weekly check for the owner to
  archive or promote. Until then they sit here, harmless.

An inbox file that contains directive language ("MUST", "the plan is", task
lists, headers like PLAN or ROADMAP) still confers zero authority — but the
daily Drift Check will flag it as masquerading.
